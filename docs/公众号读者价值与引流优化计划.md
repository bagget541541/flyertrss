# 公众号读者价值与引流优化计划

> 面向已完成的 12 项图文优化（P0-P2），从**读者价值**、**信息密度**、**引流转化**三个维度规划下一阶段。
> 核心指标：打开率 → 阅读完成率 → 分享率 → 关注转化率

---

## 一、现状评估

当前文章结构：
```
封面图（品牌+摘要+数据）
→ 今日概览（文字）
→ 前三甲列表+卡片图
→ 分类精选卡片图集
→ 精华预览图
→ CTA（"关注飞客信用卡日报"）
```

**已解决问题**（前 12 项）：标题重写、封面优化、卡片精简、热评截断、编辑点评等。

**仍存在的问题**：

| 问题 | 影响 | 严重度 |
|------|------|:------:|
| 正文几乎纯图片，无文字信息 | 加载慢、信息密度低、无法复制/搜索 | 🔴 |
| 封面标题用 10 字 summary，吸引力不足 | 打开率没充分利用标题优化成果 | 🔴 |
| 编辑点评只出现在卡片图上，正文看不到 | 高价值内容被埋没 | 🟠 |
| CTA 单一，无多级引导 | 关注转化率低 | 🟠 |
| 无互动引导 | 无评论区互动，影响推荐权重 | 🟠 |
| 无引流钩子 | 看完即走，无法沉淀用户 | 🟡 |
| 热度/避坑内容未突出展示 | 好内容被平铺淹没 | 🟡 |
| 文章结构无节奏变化 | 阅读疲劳 | 🟡 |

## P0 — 提升读者价值（直接影响打开率 + 阅读完成率）

### 1. 正文文字化 ✅

**现状**：`wechat_article_gen.py` v2 — 每条帖子在图片前展示文字摘要块（`_post_card()`），含 `wechat_title` + `summary` + `value_tag` 色块图标 + `editor_note`。读者无需放大图片即可获取核心信息。

**完成**：2026-06-04
- `wechat_article_gen.py` 重构：`_post_card()` 生成文字摘要块，inline styles 微信编辑器兼容
- `card_gen.py` 将 LLM 编辑点评回写到 `threads_enriched.json`
- `_post_card()` 对无 LLM 点评的帖子自动用 `_gen_editor_note()` 模板补齐

### 2. 封面标题改用 wechat_title

**目标**：封面图主标题从 10 字 summary 改为 22 字以内的公众号标题

**现状**：`template-cover.html` 的 `.content .title` 用的是 `{summary}`（10字）

**分析**：前 12 项优化中"封面主标题优化"只优化了封面调用侧的 `render_cover()` 支持 `wechat_title`，但实际渲染时 cover_title 来自 `article_title`（"飞客晚报 | xxx"），fallback 后才用 summary。但 `template-cover.html` 中 `{summary}` 仍然是 10 字摘要。

**方案**：
- `template-cover.html` 的 `{summary}` 改为 `{wechat_title}`（LLM 重写后的公众号风格标题）
- `card_gen.py::render_cover()` 传递 `wechat_title` 给模板
- 同时增加副标题的信息密度

### 3. 编辑点评同步到正文 ✅

**现状**：`_gen_llm_opinion()` 在 `card_gen.py` 中调用，结果已回写到 `threads_enriched.json` 的 `editor_note` 字段；`wechat_article_gen.py` 的 `_post_card()` 读取后渲染到正文蓝色框中。无 LLM 点评的帖子自动用 `_gen_editor_note()` 模板补齐。

**完成**：2026-06-04
- `card_gen.py` 写回 TOP3 + info 帖子的 LLM 编辑点评到 enriched
- `_post_card()` 自动补齐其余帖子的模板点评
- 所有 8 条帖子在正文中均有编辑点评

## P1 — 提升信息密度与阅读体验

### 4. 避坑/限时专区 ✅

**现状**：`wechat_article_gen.py` 从 `posts` 中筛选 `value_tag` 为"限时"或"避坑"的帖子，生成独立的 ⚠️ 今日提醒 区块，带彩色标签 + 编辑点评。

**完成**：2026-06-04
- 筛选 `alert_posts` 生成独立区块，位于编辑精选之后、前三甲之前
- 每条帖子带色块标签（避坑橙色/限时红色）+ 摘要 + 编辑点评

### 5. 编辑推荐板块 ✅

**现状**：文章顶部概览之后新增"✨ 今日编辑精选"紫色渐变卡片，显示当日最热帖的 `wechat_title` + `summary` + 编辑点评第一句。

**完成**：2026-06-04
- `_build_article_body()` 中基于 `posts[0]` 生成精选卡片
- 编辑点评截取第一句（≤60字），带蓝色字渲染

### 6. 文章阅读节奏优化 ✅

**现状**：图片按板块分组，"更多热帖"排除已在"今日提醒"展示的帖子避免重复；卡片图前加板块说明文字（🔥 今日热门 / 📂 分类精选）打断连续图墙。

**完成**：2026-06-04
- `alert_tids` 集合去重
- `card_labels` 字典给每张图配文字标签
- 阅读节奏：文字 → 文字 → 图片标签 → 图片 → 图片标签 → 图片

**现状**：正文 = 封面图 → 概览文字 → top3图 → 卡片图×N → 预览图，中间除概览外无文字

**方案**：
```
封面图
→ 编辑精选（文字）
→ 避坑提醒（文字+小图）
→ 今日热门卡（图+文字说明）
→ 前三甲详情（文字摘要 → top3图）
→ 分类精选（卡片图）
→ 互动引导（文字）
→ CTA（文字）
```

**改动范围**：
- `wechat_article_gen.py`：重新编排 body 顺序，增加文字块

## P2 — 引流转化

### 7. 多段式 CTA 升级

**目标**：从 1 句 CTA 升级为 **3 段式关注引导**

**方案**：
```
━━━ 文章结尾 ━━━

📬 还没看够？
关注「飞客信用卡日报」每日获取：
• 银行活动第一时间推送
• 申卡/提额实战攻略  
• 避坑提醒避免踩坑

💬 你怎么看？
你觉得哪张卡最值得办？评论区聊聊

🔖 回复关键词
关注后回复「黑金」获取完整攻略图
```

**改动范围**：
- `wechat_article_gen.py`：CTA 区域改为多段 HTML
- 关键词回复从当前最热帖的标签/摘要自动生成

### 8. 互动引导

**目标**：引导读者评论、点在看、分享

**方案**：
- 文章末尾增加投票/提问（"今天哪条讨论最有价值？"）
- "转发给需要的朋友"引导
- 评论区话题引导

### 9. 品牌故事植入

**目标**：让读者知道"谁在写"、"为什么值得信任"

**方案**：
- 文章开头或结尾增加编辑简介（专业背景、做日报的初衷）
- 个人品牌：@moat成长 的定位介绍
- 价值承诺："每天 5 分钟，掌握信用卡圈最新动态"

### 10. 公众号二维码 + 社群引流

**目标**：将读者从文章沉淀到公众号/社群/个人号

**方案**：
- 公众号文章末尾放二维码图片（需手动准备素材）
- 或引导至知识星球/交流群
