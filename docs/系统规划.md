# 飞客信用卡日报 — 系统规划

## 一、当前架构总览

```
fetcher.py          # 抓取 → threads_raw.json + threads_filtered.json
    ↓
summary.py          # 分类 + 日报渲染 → 日报_YYYY-MM-DD.{md,html,png}
    ↓
card_gen.py         # 卡片生成 → _cards/*.png (含封面、前五、top3)
    ↓
run.py              # 一键编排
```

**数据流：** 飞客论坛 HTML → 解析 → 过滤 → LLM/规则分类 → 模板渲染 → 输出
